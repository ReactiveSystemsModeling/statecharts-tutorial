Solution 4

Integrate generated traffic light statechart into Java UI

In this execise code is generated from the traffic light statechart. The generated state machine is integrated with the manually written Java application that displays a traffic light.

What is in the project

You will find additional files in this exercise project.

entry purpose
JRE System Library Ignore it. Java is used and this is an Eclipse Java project wich requires the library.
src This folder contains the hand written Java code along with the code generated from the statechart. You can take a look at it.
04-Solution-Run-UI.launch This is a launch configuration that you can use to start the UI application. Simply right-click and choosee "Run As > 04-Run-UI".
sc.sgen This is the configuration for the code generator. This is required to generate code.

The most interesting parts are the generated state machine classes in the Java package traffic.light.trafficlightctrl and the application class TrafficLightDemo which gives an example how to setr up and integrate the generated state machine code.

Run the UI

If you have implemented the state machines event listerners you can run the application. Select the file 04-Run-UI.launch in the project explorer. Right-click the entry and select "Run As > 04-Solution-Run-UI". This will launch the UI (see screenshot). You can observe how the traffic light cycles through the different phases. This UI includes elements like buttons which do not work yet but will be integrated in the next exercises.
